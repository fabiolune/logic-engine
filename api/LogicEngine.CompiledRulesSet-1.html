Remarks

This class provides functionality to apply a set of compiled rules to an item, check detailed results of rule application, and retrieve the first matching rule. It implements the IAppliable<T>, IDetailedAppliable<T, TOut>, and IAppliedSelector<TIn, TOut> interfaces.

Remarks

The CompiledRulesSet<T> class initializes a set of rules that can be applied to input data. If the provided rules array is empty, default functions are used for rule evaluation.
